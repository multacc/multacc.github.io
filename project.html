Multacc

CS 495: Spring 2020

This project started as "OneTact" before quickly rebranding.

he protecc, he attacc, but most importantly, he multacc

Our initial goals were narrow but probably more realistic in hindsight. We accomplished about half of our updated goals, which we've kept track on in our gihub project board. The sidebar contains all the links you'll need.

Achievements

As of April 23 (sprint-3 milestone), our app supports:

  • Full add/edit/delete/search functionality for contacts
  • Launching social apps from within contact details
  • Reading and sending GroupMe messages
  • Creating and sharing your contact info (Profile)

Tools

In addition to using the Github project board (which connects with issues/PRs seamlessly), we also made use of a few other tools for automating our workflows:

  • Github Actions (CI/CD): for automating tests and builds
  • Dailybot: for automating asynchronous standups
  • Todobot: for automatically converting TODOs in code to github issues
  • Slack: for notifications and bots (and for communication obviously)
